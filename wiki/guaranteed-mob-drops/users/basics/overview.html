Description

Guaranteed Mob Drops is a mod that adds a few items that are dropped by mobs, they can be combined to form the respective full drop of that mob. (Mobs still drop their regular drops)

Installation

Make sure you have Forge!

The installation is quite simple, falling into the same routine as other mods. First you want to head over to the CurseForge page. Download the latest version, make sure to check the Files tab in case CurseForge fails to update the files sidebar.

Open the /.minecraft/ folder, you can find this folder by using this official Minecraft guide.

Once you have found the /.minecraft/ folder, drag the downloaded mod file into the /mods/ folder, if this folder doesn't exist just create it. You can then run your Forge 1.12.2 profile using the Minecraft launcher.
